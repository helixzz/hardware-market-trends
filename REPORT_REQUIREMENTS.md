# 日报增强要求

每日硬件市场跟踪任务产出的日报，除了已有的价格与来源信息，还必须新增以下分析层。

## 1. 趋势判断

对每个大类至少给出：
- 今日覆盖率
- 相对近 7 天的覆盖率变化
- 相对昨日的价格变化方向
- 当前趋势结论：上涨 / 下跌 / 持平 / 波动扩大 / 暂无判断

## 2. 采购建议

对以下关键维度给出建议：
- DDR5-6400 RDIMM
- DDR5-5600 RDIMM
- PCIe4 TLC
- PCIe5 TLC
- PCIe4 QLC

采购建议格式必须包含：
- 结论：立即采购 / 等待 / 分批采购 / 仅应急采购 / 暂无依据
- 依据：价格位置、覆盖率、来源质量、近期波动
- 风险：样本稀疏、来源塌缩、异常价扰动、产品口径分裂

## 3. 缺口说明

日报中必须明确写出：
- 仍为空的关键容量位
- 是哪种缺失原因
- 是否值得后续重点补源

## 4. 今日新增信息

如果今天相较昨天有新增来源、新增产品家族或新增有效样本，必须单列说明。

## 5. 低置信度纪律

如果趋势或采购建议证据不足，必须明确写“暂无足够依据”，不能为了看起来完整而强行给判断。
